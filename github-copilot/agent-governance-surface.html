When single-surface governance matters

  • Regulated industries. Auditors prefer one log to subpoena, not five.
  • Incident response. One query to reconstruct "what did the agent touch?" beats four.
  • Access revocation. Disabling a first-party bot is one toggle; off-boarding four vendors is four tickets.

Where this story gets complicated

  • Copilot's content-exclusion gap is real. The one place Copilot's governance is worse than the alternatives today.
  • Claude Code Action runs on your runner. This means code never leaves your GitHub control plane — the cleanest third-party story.
  • "Mixed" isn't inherently bad. Vendor-side usage views often have better product-specific analytics than GitHub's generic ones.
  • All four tools change governance surfaces quarterly. Re-check this table every 90 days.
