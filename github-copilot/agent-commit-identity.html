Who authored this PR?

Agent-generated PRs have an author identity. Whether that author is a first-party GitHub bot or a third-party OAuth App changes which rulesets apply, which audit logs fire, and which procurement conversations you need to have.

What this doesn't say

  • Third-party doesn't mean worse. A custom GitHub App gives you more permission control than a first-party bot. The cost is setup, not security.
  • Copilot has its own gaps. The content-exclusion limitation above is a real one — Copilot cloud agent does not honor Copilot content exclusions.
  • Custom Apps exist. Claude Code Action supports a self-owned GitHub App with your chosen name; use it if org policy blocks the vendor-owned App.

About this infographic

A platform-engineering view of commit-author identity for AI coding agents on GitHub. The claim is narrow: first-party bot identity reduces governance setup (rulesets, audit, procurement) but does not make an agent inherently more or less safe.
